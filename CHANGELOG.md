## [v0.1.1](https://github.com/irl-llc/git-spice-code-extension/releases/tag/vv0.1.1) - 2026-06-04
### Fixed
- Fixed the stack view occasionally loading blank: the webview now announces readiness only after its message listener is attached, so the extension's initial state push can no longer be dropped by a render-timing race.

## [v0.1.0](https://github.com/irl-llc/git-spice-code-extension/releases/tag/vv0.1.0) - 2026-06-03
### Added
- Show change-request (PR/MR) status badges — Open, Merged, or Closed — on each branch card, fetched from the forge (`gs ll -S`) alongside comment counts. The `git-spice.showCommentProgress` setting is renamed to `git-spice.showRemoteForgeStatus`, which now gates both comment counts and change-request status; re-enable it after upgrading.
- Integration-branch support (beta): renders the git-spice integration branch atop the stack with its build status and a marigold "rebuild" state, marks branches outside the integration build, and lets you add/remove branches from the integration tip list via the branch context menu. Requires a git-spice build that provides the `integration` command.
